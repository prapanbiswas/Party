How to Use LuminaParty
1
Host a Party

Tap Host Party on the landing screen. A 4-digit room code appears at the top — this is your party ID.

2
Share the Code

Send the code via WhatsApp or SMS. Friends open prapanbiswas.github.io/Party, tap Join Lights and type your code.

3
Their Screen = A Light

Each joined phone becomes a Light Node — its screen fills with synchronized color from the host. More phones = bigger show.

4
Enable Music Sync

Go to the Audio tab, tap Enable Mic Sync and allow mic access. Lights now react to music in real time.

Best result: play music loud on a speaker, host phone near the speaker, guests hold phones around the room.

Sound Activated Disco Lights — 31-Band Real-Time Audio Analysis

LuminaParty's audio engine is built on a professional 19-module DSP pipeline that analyzes microphone input in real time. This system powers genuine sound activated disco lights online behavior — the lights do not simply react to overall volume, but specifically detect sub-bass frequencies (20–80 Hz), bass (80–250 Hz), mid-range (250–4 kHz), and high frequencies (4–20 kHz) independently. The 31-band ISO third-octave spectrum analyzer, BPM tracker, pitch detector, and musical key detector all run simultaneously. Bass hits trigger white flash bursts. BPM-synced Hue Rotate mode spins the color wheel at exactly the tempo of the music. These are genuinely music reactive lights online — not a simple volume meter.

Six Audio-Reactive Mapping Modes for Beat Sync Light

Six distinct audio reactive modes allow DJs and party hosts to customize how the beat sync light browser system maps music to visuals. Spectrum Map assigns each device to a different frequency band. Bass Color shifts hue based on spectral centroid — warmer colors for bass-heavy music, cooler for treble-dominant tracks. Beat Flash delivers a white burst on every detected beat. Psychedelic mode builds energy progressively. Energy Pulse drives brightness from combined loudness. Hue Rotate spins the color wheel at BPM. Together, these make LuminaParty the most feature-rich bpm light sync online tool available without installing an application.

Multi-Device Party Light Sync — Host and Join System

The multi device party light synchronization system is built on PeerJS WebRTC connections. The host generates a unique 4-digit room code. Clients join by entering this code on the same LuminaParty page. Once connected, the host broadcasts color frames at 40 frames per second to all connected nodes. Each client receives a packet containing the full color array, extracts its assigned node color, and applies it to the screen with a 15-millisecond CSS transition for buttery smoothness. This system enables sync lights online free across an unlimited number of devices on any network, making it ideal for crowd lighting, party decorations, and synchronized screen displays without any paid subscription or local network party light hardware.
